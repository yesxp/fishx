# 鱼渔娱 (FishX)

钓鱼人的一站式轻量工具。

## 技术栈

- Uni-app 4.66 (Vue 3 + TypeScript + Vite)
- Pinia 状态管理
- 和风天气 API
- 腾讯云图像识别

## 项目结构

```
src/
├── pages/
│   ├── index/        # 首页（渔获瀑布流）
│   ├── weather/      # 天时（钓鱼指数）
│   ├── map/          # 钓点（地图）
│   ├── mine/         # 我的（个人中心）
│   ├── login/        # 登录
│   └── catch/        # 渔获（创建/详情）
├── components/       # 公共组件
├── stores/           # Pinia 状态
├── api/              # 接口封装
├── utils/            # 工具函数
└── static/           # 静态资源
```

## 功能模块

| 模块 | 功能 |
|------|------|
| 首页 | 天气指数条 + 标签筛选 + 双列瀑布流 |
| 天时 | 钓鱼指数 + 气压趋势 + 最佳时段 + 鱼口预测 |
| 钓点 | 地图搜索 + 钓点标记 + 打卡 |
| 我的 | 统计 + 渔获/钓点/收藏 + 设置 |
| 登录 | 微信授权 / 手机号验证码 |
| 渔获 | 拍照 + AI识鱼 + 评论点赞 |

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev:h5

# 构建
npm run build:h5
```
