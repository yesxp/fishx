## Story 1: 首页渔获瀑布流 🏠首页

**优先级:** P0  
**复杂度:** M  
**依赖:** Story 5（需要渔获数据）

作为 **钓鱼人**  
我想 **打开 App 看到附近/最新的渔获**  
以便 **受到激励、发现好钓点**

### 验收条件

- [ ] AC1: 打开首页 → 3 秒内加载渔获瀑布流（双列交错）
- [ ] AC2: 每张卡片显示：渔获照片 + 鱼种 + 钓点名 + 时间 + 天气小标
- [ ] AC3: 卡片右下角显示 ❤️ 点赞数 + 💬 评论数
- [ ] AC4: 下拉刷新 + 滚动到底自动加载更多（每页 20 条）
- [ ] AC5: 点击卡片 → 展开渔获详情（大图 + 完整信息 + 评论列表）
- [ ] AC6: 空态（无渔获）→ 显示引导 "去钓鱼吧！" + 拍照入口

## Story 2: 首页钓鱼指数 🏠首页

**优先级:** P0  
**复杂度:** M  
**依赖:** 无

作为 **钓鱼人**  
我想 **在首页快速看到今天钓鱼指数**  
以便 **判断今天适不适合出钓**

### 验收条件

- [ ] AC1: 首页顶部显示今日钓鱼指数 0-100 + 一句话建议
- [ ] AC2: 指数颜色随分数变化（绿/黄/橙/红）
- [ ] AC3: 显示定位城市名
- [ ] AC4: 点击指数区域 → 跳转天时详情页
- [ ] AC5: 未授权定位 → 提示选择城市
- [ ] AC6: 网络异常 → 显示缓存数据 + 提示

---

## Story 3: 天时详情 🌤️天时

**优先级:** P0  
**复杂度:** M  
**依赖:** Story 2

作为 **钓鱼人**  
我想 **查看详细天气和最佳出钓时段**  
以便 **选对时间出门**

### 验收条件

- [ ] AC1: 显示大字钓鱼指数 + 结论（"适合出钓" / "建议改天"）
- [ ] AC2: 气压折线图（过去 12h + 未来 12h）
- [ ] AC3: 核心数据：气温 / 气压 / 风力 / 天气 / 水温
- [ ] AC4: 最佳出钓时段（1-2 个窗口，颜色标记）
- [ ] AC5: 逐小时鱼口强度条形图
- [ ] AC6: 底部推荐 "这个天气下钓友们的渔获"（3-6 张卡片）
- [ ] AC7: 气压骤降 >5hPa/3h → 警告提示

## Story 4: 钓点地图 🗺️钓点

**优先级:** P0  
**复杂度:** L  
**依赖:** Story 5

作为 **钓鱼人**  
我想 **在地图上看到附近钓点和最近渔获**  
以便 **选择去哪钓**

### 验收条件

- [ ] AC1: 打开地图 → 以定位点为中心，显示钓点标记
- [ ] AC2: 每个标记是**渔获照片气泡**（最近一张渔获缩略图）
- [ ] AC3: 点击标记 → 弹出卡片：钓点名 / 评分 / 渔获数 / 距离
- [ ] AC4: 卡片内可跳转 → 钓点详情页（渔获时间线 + 评分 + 数据）
- [ ] AC5: 长按空白处 → "标记钓点"面板（名称 + 类型 + 备注）
- [ ] AC6: 支持搜索钓点（名称搜索）
- [ ] AC7: 空态 → 提示"标记第一个钓点"
- [ ] AC8: 支持切换标准/卫星底图

## Story 5: 渔获拍照 + AI 识鱼 + 拍照即发布 📷记录

**优先级:** P0  
**复杂度:** L  
**依赖:** 无

作为 **钓鱼人**  
我想 **拍一张鱼的照片，自动识别并发布**  
以便 **轻松记录渔获，同时分享给钓友**

### 验收条件

- [ ] AC1: 点击底部 "+" → 选择拍照/相册
- [ ] AC2: 选中后 3 秒内返回 AI 识别结果（鱼种 + 置信度）
- [ ] AC3: 自动填充：时间 / GPS 地点 / 当前天气
- [ ] AC4: 用户可修改鱼种（下拉选择 Top 20 常见鱼种）
- [ ] AC5: 置信度 <70% → 提示"识别不确定，请确认"
- [ ] AC6: 保存 → 渔获照片**默认公开**（出现在首页流 + 钓点气泡）
- [ ] AC7: 提供"仅自己可见"开关（隐私选项）
- [ ] AC8: 图片压缩 <500KB 再上传
- [ ] AC9: 无网络 → 本地暂存，有网后自动上传

## Story 6: 渔获详情 + 互动

**优先级:** P0  
**复杂度:** M  
**依赖:** Story 5

作为 **用户**  
我想 **查看渔获详情并互动**  
以便 **了解钓鱼信息、表达认可**

### 验收条件

- [ ] AC1: 渔获详情页：大图 + 鱼种 + 天气 + 地点 + 时间 + 钓点链接
- [ ] AC2: 点赞按钮（❤️ 点击切换，实时更新计数）
- [ ] AC3: 评论输入框（≤100 字）+ 评论列表（时间倒序）
- [ ] AC4: 分享按钮（生成图片卡片 / 复制链接）
- [ ] AC5: 查看钓点 → 跳转钓点详情页
- [ ] AC6: 查看作者主页 → 跳转个人主页

## Story 7: 钓点详情 🗺️钓点

**优先级:** P0  
**复杂度:** M  
**依赖:** Story 4

作为 **钓鱼人**  
我想 **了解一个钓点的完整信息**  
以便 **决定去不去这里**

### 验收条件

- [ ] AC1: 钓点详情页：名称 + 类型 + 评分 + 总渔获数 + 距离
- [ ] AC2: 渔获时间线（按时间倒序，瀑布流展示该钓点的所有渔获）
- [ ] AC3: 钓点环境信息（如有：水质、深度、收费）
- [ ] AC4: "打卡"按钮（标记"我来过"）
- [ ] AC5: "导航"按钮（调起地图导航）
- [ ] AC6: "分享"按钮

## Story 8: 个人中心 👤我的

**优先级:** P0  
**复杂度:** S  
**依赖:** Story 5

作为 **用户**  
我想 **管理我的个人资料和渔获**  
以便 **回顾钓鱼历程**

### 验收条件

- [ ] AC1: 显示头像 + 昵称
- [ ] AC2: 统计卡片：总渔获数 / 最常钓的鱼 / 去过几个钓点
- [ ] AC3: "我的渔获"列表（时间线）
- [ ] AC4: "我的钓点"列表（收藏/标记的）
- [ ] AC5: 设置：定位授权、通知、清除缓存、关于
