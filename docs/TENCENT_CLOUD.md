# 腾讯云开发配置

## 环境ID

在腾讯云开发控制台获取环境ID，替换以下文件中的 `YOUR_ENV_ID`：

- `src/lib/cloudbase.ts`

## 数据库集合

在腾讯云开发控制台创建以下集合：

1. **spots** - 钓点表
2. **catches** - 渔获表
3. **reviews** - 评价表

## 云函数

在腾讯云开发控制台部署以下云函数：

1. **getSpots** - 获取钓点列表
2. **createSpot** - 创建钓点
3. **updateSpot** - 更新钓点

## 权限设置

在腾讯云开发控制台设置数据库权限：

- **spots**: 所有用户可读，仅创建者可写
- **catches**: 所有用户可读，仅创建者可写
- **reviews**: 所有用户可读，仅创建者可写

## 部署步骤

1. 登录腾讯云开发控制台
2. 创建环境（选择上海/广州区域）
3. 创建数据库集合
4. 部署云函数
5. 设置数据库权限
6. 将环境ID填入 `src/lib/cloudbase.ts`
